# 중증외상센터 캐릭터 테스트

드라마 "중증외상센터"의 캐릭터들 중 나와 가장 잘 맞는 캐릭터를 찾아보는 퀴즈 웹 애플리케이션입니다.

## 주요 기능

- 사용자의 성향을 파악하는 다중 선택 질문
- 실시간 진행 상태 표시
- 결과에 따른 캐릭터 매칭
- 모바일 친화적인 반응형 디자인
- 결과 공유 기능

## 기술 스택

- HTML5
- CSS3 (Flexbox, Grid, 반응형 디자인)
- JavaScript (ES6+)
- GitHub Pages (호스팅)

## 로컬에서 실행하기

1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/[사용자명]/trauma-center-quiz.git
   ```

2. 프로젝트 디렉토리로 이동:
   ```bash
   cd trauma-center-quiz
   ```

3. 로컬 서버를 실행합니다 (예: Python의 경우):
   ```bash
   python -m http.server 8000
   ```

4. 브라우저에서 `http://localhost:8000`을 엽니다.

## 기여하기

1. 이 저장소를 포크합니다.
2. 새로운 브랜치를 생성합니다: `git checkout -b feature/새로운기능`
3. 변경사항을 커밋합니다: `git commit -m '새로운 기능 추가'`
4. 포크한 저장소에 푸시합니다: `git push origin feature/새로운기능`
5. Pull Request를 생성합니다.
